Introduction

In this Article you can see our API Backend documentation to connect to our backend Server.

The default api endpoint is: http://localhost:4200/api/v1/mountains

CREATE

To create a new Database-Entry for a mountain just make a http post calls to the default endpoint. The http body should contain the JSON body containing the following fields:

  • "name"
  • "location"
  • "length"
  • "imageUrl"
  • "maxHight"

Example:
POST http://localhost:4200/api/v1/mountains/ with body=json_object

READ

You can either read the hole dataset by making a http get request to the default endpoint. Or, if you want a specific entry, append the corresponding id the the url.

Example:
GET http://localhost:4200/api/v1/mountains
or:
GET http://localhost:4200/api/v1/mountains/2

UPDATE

To update a specific entry make a http put calls to the endpoint with the corresponding id as suffix. The html body should still contain the json object. With the updated values.

Example:
put http://localhost:4200/api/v1/mountains/2 with body=json_object

DELETE

To delete an entry just make a http delete request to the default endpoint and append the corresponding id of the entry.

Example:
delete http://localhost:4200/api/v1/mountains/2
